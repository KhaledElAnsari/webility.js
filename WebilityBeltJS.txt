the Web Utility Belt

Features:
✓ - title() // like the python function
✓ - quote() // deprecated from Javascript
✓ - capitalize() // uppercase first letter
✓ - browser() // returns the current browser
✓ - UUID() // like the python UUID function, it creates and check if the string is a valid UUID
- dateFormat() // returning a specific date format
- toUTC() // return convert Date object to UTC
✓ - today() // returns today's date
✓ - objToArray() // from objects with values to arrays of objects
✓ - sort() // it can sort arrays and objects
✓ - getDays() // list of between two dates
✓ - dayAfter() // get the date after (for example) 9 days
✓ - isEmpty() // check if object is empty
✓ - get() // for the dictionary just like python
✓ - pageready() // return true when the page content is 100% ready
✓ - ismobile() // return true when the device is a mobile one
✓ - device() // return the type of the device (Desktop, Tablet or Mobile)
✓ - os() // return the name of the user's OS
✓ - select() // css selector for the elements
✓ - ID()
✓ - addclass()
✓ - hasclass()
✓ - removeclass()
- deepClone() // deep cloning Objects
